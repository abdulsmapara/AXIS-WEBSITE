Problem Statement

This is a 5 hr programming paradise for insomniacs where they will be served with mind boggling algorithmic questions and need to solve them up. There will be flexibility to use any programming language of their own choice. A week before there will be warm up challenge, an all-day coding event for those who want to brush up their coding skills.
Stay tuned for warm up dates!!

Rules and Regulations

  • Participants must register themselves for INSOMNIA under www.axisvnit.org before participating in any of the events.
  • The Contest consists of 7 programming questions.
  • The participant will score maximum points on correctly solving all the test cases.
  • Partial marking will be there in case the participant correctly solves some of the test cases.
  • Maximum marks for a question may vary.
  • Participants need to complete the contest between 10pm - 3am.
  • Insomnia Warmup to be conducted for the sake of practice for the participants. The time for the same is 11 am to 11 pm. The points scored in the practice contest will NOT be considered in the Main Event.
  • NOTE: In case of any disputes, the decisions taken by the AXIS Executive Committee will be bound and final.
